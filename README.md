# Running Astrometrica on Apple Silicon macOS

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21153650.svg)](https://doi.org/10.5281/zenodo.21153650)

**A validated workflow for running Astrometrica on Apple Silicon (M1–M4) Macs for the International Astronomical Search Collaboration (IASC).**

Astrometrica is a legacy Windows application widely used for asteroid detection and astrometric measurements. This repository provides a complete, tested solution using **Sikarugir** (Wine wrapper) and **Rosetta 2**, enabling asteroid search campaigns on modern Apple Silicon hardware.

The workflow has been independently verified with official IASC practice image sets.

## Features
- Complete installation & configuration for Apple Silicon
- Detailed troubleshooting for common issues
- Astrometric data reduction and plate solving
- Blink comparison and manual moving object detection
- Known object overlay and MPC report generation
- Fully validated with official IASC practice datasets
- Tested on: **macOS 15.7.4 + Apple M1**
