# Running Astrometrica on Apple Silicon macOS

Astrometrica is a Windows-only application widely used for asteroid detection.  This project documents a validated workflow for running Astrometrica on Apple Silicon Macs using Sikarugir and Rosetta 2.  The workflow has been verified using official IASC practice datasets.
 
**A Verified Workflow for the International Astronomical Search Collaboration (IASC)**

This repository contains the full guide for running the legacy Windows software **Astrometrica** on modern Apple Silicon Macs (M1/M2/M3/M4) using Sikarugir + Rosetta 2.

## Features
- Complete installation instructions for macOS
- Troubleshooting guide
- Full IASC workflow validation including Configuration, Plate solving, Blink comparison, Candidate detection and MPC report generation
- Tested on macOS 15.7.4 + M1
