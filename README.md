**Running Astrometrica on Apple Silicon macOS**

Astrometrica is a Windows-only application widely used for asteroid detection. This repository contains a validated workflow for running Astrometrica on Apple Silicon Macs (M1–M4) using Sikarugir and Rosetta 2. The workflow has been verified using official International Astronomical Search Collaboration (IASC) practice datasets.

## Features
- Complete installation instructions for macOS
- Configuration and troubleshooting guide
- Plate solving and image calibration
- Blink comparison and moving object detection
- Minor Planet Center (MPC) report generation
- Validated using official IASC practice datasets
- Tested on macOS 15.7.4 (Apple M1)
